# AS5600 Driver for STM32 (HAL)

A simple, lightweight AS5600 magnetic absolute encoder driver for STM32 microcontrollers using the STM32 HAL layer.

The driver provides APIs for:

- querying angular position (PWM, I2C),
- configuring the encoder,
- querying status and errors,
- and useful helper functions for motor control, such as **tracking absolute angular position**.

---

## Dependencies

- **STM32 HAL**
  - HAL_TIM (PWM input capture)
  - HAL_I2C

---

## Supported Hardware and Environment

- AS5600 magnetic absolute encoder
- STM32 microcontrollers (HAL-based projects)

---

## Feature Overview

| Feature | Supported | Not Supported |
|------|----------|---------------|
| Angular Position Readout | I2C<br>PWM | Analog |
| Position | Absolute Angular Position Tracking | Angular Velocity Calculation |
| Error Handling | Error callback for various errors | – |
| Status Readout | Magnet too strong<br>Magnet too weak<br>Magnet detected | – |
| Configuration | Power mode<br>Hysteresis<br>Output stage<br>PWM frequency<br>Slow filter<br>Fast filter threshold<br>Watchdog | Burn (min/max angle range setting) |

> **Note:**  
> The *Burn* function in the AS5600 represents permanent settings, which is why the driver intentionally does not support it.

---

## Quick Start

#### Peripherals:

> 1. Initialize I2C peripheral (Standard / Normal mode)
> 2. Initialize TIM peripheral in PWM input capture mode

#### Code Steps:

> 1. Include driver header  
> 2. Create encoder handle  
> 3. Initialize encoder  
> 4. Attach peripherals  
> 5. Configure encoder  

---

## Example Code

> **Note:**  
> The example assumes a project structure generated by STM32CubeMX.

```c
#include "AS5600Driver.h"

AS5600Handle_Typedef *Encoder1;

int main(void)
{
  HAL_Init();
  SystemClock_Config();

  MX_GPIO_Init();
  MX_I2C1_Init();
  MX_TIM2_Init();

  /* Create encoder */
  Encoder1 = AS5600_Create(0x36, 360.0f, 0.0f);

  /* Attach peripherals */
  AS5600_AttachPeripheral(Encoder1, &hi2c1, &htim2);

  /* Configure encoder */
  AS5600_Configure(Encoder1,
                   NOM,
                   OFF,
                   PWM,
                   _115Hz,
                   _16x,
                   SlowFilterOnly,
                   Off);

  /* Update status */
  AS5600_UpdateStatus(Encoder1);

  while (1)
  {
    /* Read angular position via PWM */
    AS5600_ReadAngle_PWM(Encoder1);

    /* Update absolute angular position */
    AS5600_UpdateAbsolutePosition(Encoder1);
  }
}
```

## Error Handling

Error handling can be controlled via the define value found in the AS5600Driver_Config.h file.

Error handling active:
```c
#define config_USE_ERROR_HANDLING	  1
```

Error handling inactive:
```c
#define config_USE_ERROR_HANDLING	  0
```

When error handling is active, the AS5600_ErrorHandler() function is called. Its current implementation is:
```c
__attribute__((weak)) void AS5600_ErrorHandler(AS5600Handle_Typedef *pAS, eInfo error)
{
	switch(error)
	{
	case AS5600_OK:
		while(1);
		break;

	case AS5600_NULL_POINTER:
		while(1);
		break;

	case AS5600_CALLOC_FAIL:
		while(1);
		break;

	case AS5600_I2C_COMM_ERROR:
		while(1);
		break;

	case AS5600_PWM_MODE_NOT_INITIALIZED:
		while(1);
		break;

	case AS5600_INPUT_PWM_FREQ_ERROR:
		//while(1);
		break;


	case AS5600_INPUT_PWM_DUTYCYCLE_ERROR:
		while(1);
		break;

	case AS5600_DEVIDE_BY_ZERO:
		while(1);
		break;

	case AS5600_INVALID_MAX_ANGLE:
		while(1);
		break;

	case AS5600_INVALID_MIN_ANGLE:
		while(1);
		break;

	case AS5600_INVALID_MIN_MAX_ANGLE:
		while(1);
		break;

	case AS5600_OTHER_ERROR:
		while(1);
		break;


	}
}
```

Thanks to the weak attribute, it is possible to implement the error handler function individually. Here is an example (main.c):
```c
void AS5600_ErrorHandler(AS5600Handle_Typedef *pAS, eInfo error)
{
	switch(error)
	{
	case AS5600_I2C_COMM_ERROR:
		if (HAL_I2C_IsDeviceReady(&hi2c1, (0x36 << 1), 4, 1000) == HAL_OK)
                   {
                   uart_printf(&huart1, "Device OK");
                   }
                   else
                   {
                   uart_printf(&huart1, "Device NOT available");
                   }
		break;


	case AS5600_INPUT_PWM_FREQ_ERROR:
		uart_printf(&huart1, "Wrong PWM settings");
		break;



	}
}
```

# AS5600 driver STM32-höz (HAL)

Egyszerű, lightweight AS5600 mágneses abszolút enkóder driver STM32 mikrokontrollerekhez, STM32 HAL réteg használatával.

A driver API-kat biztosít:

- szöghelyzet lekérdezésére (PWM, I2C),
- az enkóder konfigurálására,
- státusz és hibák lekérdezésére,
- valamint motorvezérléshez hasznos segédfunkciókra, például **abszolút szögpozíció számontartására**.

---

## Függőségek

- **STM32 HAL**
  - HAL_TIM (PWM input capture)
  - HAL_I2C

---

## Támogatott hardver és környezet

- AS5600 mágneses abszolút enkóder
- STM32 mikrokontrollerek (HAL alapú projektek)

---

## Funkcióáttekintés

| Funkció | Támogatott | Nem támogatott |
|------|----------|---------------|
| Szöghelyzet lekérdezés | I2C<br>PWM | Analóg |
| Pozíció | Abszolút szögpozíció számontartása | Szögsebesség számítás |
| Hibakezelés | Error callback különböző hibákra | – |
| Státusz lekérdezés | Magnet too strong<br>Magnet too weak<br>Magnet detected | – |
| Konfigurálás | Power mode<br>Hysteresis<br>Output stage<br>PWM frequency<br>Slow filter<br>Fast filter threshold<br>Watchdog | Burn (min/max szögtartomány beállítás) |

> **Megjegyzés:**  
> A *Burn* funkció az AS5600-ban permanens beállítást jelent, ezért a driver szándékosan nem támogatja.

---

#### Perifériák:

> 1. I2C periféria inicializálása (Standard / Normal mode)
> 2. TIM periféria inicializálása PWM input capture módban

#### Kód lépések:

> 2. Enkóder handle létrehozása  
> 3. Enkóder inicializálása

## Hibakezelés

A hibakezelés AS5600Driver_Config.h fájlban található define értékével állítható.

Hibakezelés aktív:
```c
#define config_USE_ERROR_HANDLING	  1
```

Hibakezelés inaktív:
```c
#define config_USE_ERROR_HANDLING	  0
```
